TraitSharp provides zero-allocation trait views over contiguous arrays using Unsafe.AddByteOffset strided access. These benchmarks compare TraitSpan access patterns against native Span<T> baselines performing identical work.

Each comparison group isolates a single variable: the access mechanism. The native baseline and trait variant execute the same inner loop, sum the same fields, over the same data. Ratios are computed against the group-specific baseline (not the BenchmarkDotNet class baseline).
